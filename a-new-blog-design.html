A New Blog Design

By John Mathews
Category: Web
December 2020

The blog has a new design! Out with the old, and in with well-written HTML, an improved CSS framework, maintainable code, dark mode, and articles with multiple categories.

This was my first ever project using HTML and CSS, and the codebase for the original blog was terrible. It was poorly written and hard to maintain.

As I’ve learnt more about web development, what was once difficult and confusing has become intuitive and familiar. As I become more comfortable with “good” dev work, working with this blog became increasingly uncomfortable. I wanted to update the blog so that it would be easy and fun to use again. I want to be able to play with it quickly and efficiently if necessary.

I wasn’t aiming for a radical re-design, I like that the focus is on text and I’m not exploring any on-trend design choices. I think my original design choices have help up well. I want a design that will work for many years, with HTML is easier to read, and design elements that are easier to work with.

I hope that I’ll be writing here more regularly over the next few months. It’s been a busy year and there are posts to be written!
