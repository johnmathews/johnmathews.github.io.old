A New Blog Design

By John Mathews
December 2020

We have a new blog design! This blog was my first ever project using HTML and CSS, and the code was terrible. It was poorly written and hard to maintain.

I’ve learnt a lot about web development this year, and what was once difficult and confusing has become intuitive and familiar. I wanted to update the blog so that it would be easy and fun to use. I want to be able to play with it quickly and efficiently if necessary.

I’m not aiming a radical re-design, I like that the focus is on text and I’m not exploring any on-trend design choices. I want a design that will work for many years. The main changes are that the HTML is easier to read, and instead of using Bootstrap v3 I’m using the more intuitive Tailwind v2.0. I hope that I’ll be writing here more regularly over the next few months.

It’s been a busy year and there are posts to be written!
